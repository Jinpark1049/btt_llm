# btt_llm

report parser with ollama/ langchain/ streamlit (for demo)

## Getting Started

### Ollama
For linux
```bash
curl -fsSL https://ollama.com/install.sh | sh

ollama pull gemma2:27b
```
### Streamlit
For linux
```bash
cd app
streamlit run app.py
```
